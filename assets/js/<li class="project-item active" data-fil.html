  • Ft_printf

    • The Libft project at 42 consists of recreating a standard C library by implementing commonly used functions for handling strings, memory, numbers, and linked lists. It also includes data structures such as singly linked lists (t_list) and sometimes dynamic arrays to efficiently manage data sets. The goal is to gain a deep understanding of memory management using malloc and free, as well as pointer and structure manipulation. This library will serve as a fundamental base for future 42 projects, enabling more optimized and reusable code.
